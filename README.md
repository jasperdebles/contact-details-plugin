# Contact Details Plugin

Adds professional contact information to the bottom-right corner of your WordPress site footer. Always visible, mobile-responsive, and click-to-email ready.

![Demo Footer Contact Box](screenshot.png)

## ✨ Features

- **Fixed position** - Always visible (doesn't scroll away)
- **Clickable email** - `mailto:info@ushba.nl` link
- **Professional styling** - WP-blue theme colors + shadow
- **Mobile responsive** - Perfect on all devices
- **Lightweight** - Pure PHP, no dependencies

## 🚀 Installation

### Option: WordPress Admin (Recommended)
1. Download `contact-details-plugin.zip`
2. WP Admin → **Plugins** → **Add New** → **Upload Plugin**
3. Upload ZIP → **Install Now** → **Activate**
